## Memory Review Checklist

- Does every line help the AI behave differently?
- Is it durable enough to reuse across future conversations?
- Is it safe to store?
- Are technical preferences separated from role and collaboration preferences?
- Are "ask first" and "do not store" boundaries explicit?
- Could this fit into a smaller memory field if needed?
