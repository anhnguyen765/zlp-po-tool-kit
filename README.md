# ZLP PO Tool Kit

Personal Codex skills and agent workflow assets.

## Skills

### onboard-personal-memory

Step-by-step onboarding workflow for interviewing a person and turning the answers into portable personalised memory for AI agents such as Codex, Claude, ChatGPT, Antigravity, and other assistants.

Install with Codex:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py --repo anhnguyen765/zlp-po-tool-kit --path skills/onboard-personal-memory
```
