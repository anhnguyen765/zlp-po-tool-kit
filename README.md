# ZLP PO Tool Kit

Product Owner tools, agent instructions, and workflow assets.

## Agents

### User Interview Copilot

ChatGPT agent instructions for a Product Owner interview copilot. The agent supports three phases: pre-interview preparation, live voice/transcript follow-up support, and post-interview synthesis into Confluence-ready Markdown.

Agent files: `agents/user-interview-copilot/`

## Skills

### onboard-personal-memory

Step-by-step onboarding workflow for interviewing a person and turning the answers into portable personalised memory for AI agents such as Codex, Claude, ChatGPT, Antigravity, and other assistants.

Install with Codex:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py --repo anhnguyen765/zlp-po-tool-kit --path skills/onboard-personal-memory
```
